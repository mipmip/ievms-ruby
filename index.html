Ievms-ruby

Ruby gem for provisioning Windows boxes made by ievms.sh

View on GitHub Download .zip Download .tar.gz

Ruby interface for boxes made by ievms.sh. Use this Library to provision your IE boxes from https://modern.ie.

Next to ievms.sh, ievms-ruby also works great in combination with iectrl.

Ievms-ruby is maintained by mipmip.
